FRED API キーを設定

米国の月次リアルタイムデータを取得するために、 無料の FRED API キーが必要です。
日本のデータは World Bank API（キー不要）から自動取得します。

1
fredaccount.stlouisfed.org で無料アカウント作成
2
ログイン後「API Keys」→「Request API Key」をクリック
3
発行された 32文字のキーを下に貼り付けて「開始」
⚠ キーが無効です。32文字の英数字を確認してください。
キーはこのブラウザの localStorage に保存されます。
外部サーバーには一切送信されません。
